Tutorial¶

1.1 Quickstart¶

As a simple first example, we will use ELLIE to deliver a target pixel file (TPF) and light curve for a given TIC target.

In [2]:

import ellie

First we load up the data for TIC 198591153, a relatively bright star (TESS mag = 9.0) observed in Sector 1. ELLIE is able to extract a light curve easily and automatically for such a bright, unblended object.

In [3]:

dp = ellie.data_products(tic=198591153)

Now that we have a data product for our target, we simply call the individual_tpf() function, which will extract a target pixel file, perform aperture photometry with an automatically optimized aperture choice, and do some simple systematics corrections.

In [4]:

tpf_fn = dp.individual_tpf()
dp.plot()
