ELLIE: simple light curve extraction from TESS FFIs
===================================================
.. image:: _images/logo.gif
	:height: 300px
	:width: 600px
	:align: center
	
ELLIE is an open-source python framework for downloading, analyzing, and visualizing data from the `TESS`_ Full Frame Images.
